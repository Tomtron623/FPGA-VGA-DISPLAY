# FPGA-VGA-DISPLAY
Reads an image from onboard memory, and displays it through the onboard VGA of the DE-10 Lite Board.
Coded fully in VHDL, Flashed and Optimized for Quartus (Important for ROM)
Testbenches folder contains all testbenches for individual components.
Components folder includes everything neccessary to flash the program and all the code.

Quartus Archive file is provided for easy running and compiling.
